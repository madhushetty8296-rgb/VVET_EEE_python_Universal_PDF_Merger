# Multi File PDF Merger

## Features

- Upload Multiple Files
- Merge PDF
- Convert DOCX to PDF
- Convert Images to PDF
- Download Merged PDF
- Preview All Pages

## Supported Files

- PDF
- DOCX
- PNG
- JPG
- JPEG

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```

Deploy directly on Streamlit Community Cloud.
